# 🚨 Rua Segura

O **Rua Segura** é um aplicativo de segurança urbana baseado em dados e geolocalização, desenvolvido como parte da disciplina de **Engenharia de Software II**.

O sistema tem como objetivo apoiar usuários na tomada de decisões mais seguras no contexto urbano, por meio da coleta, análise e visualização de informações relacionadas à segurança pública.

---

## 📌 Objetivo

Desenvolver uma aplicação capaz de:

- Mapear ocorrências de risco em regiões urbanas
- Fornecer informações baseadas em localização
- Apoiar decisões de deslocamento mais seguras
- Contribuir para uma percepção mais precisa da segurança local

---

## 💡 Proposta de Solução

O Rua Segura propõe:

- Uso de geolocalização para análise contextual
- Coleta colaborativa e/ou integração com dados externos
- Visualização de ocorrências em mapa
- Base para futuras integrações com mobilidade urbana e dados públicos
